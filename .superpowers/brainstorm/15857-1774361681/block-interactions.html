Block Interactions

A — Subtle & Airy

Generous spacing between blocks. Barely-visible chrome when idle. Hover shows a whisper-thin border and fades in handles. Active gets a soft glow. Block type label appears on hover. Feels like writing on a clean page.

⠿
An idle block — always has a faint background card and border. You can always see where one block ends and another begins.
⠿
⚙
Hovered. Background slightly brightens, subtle shadow lifts. Blocks feel like physical cards you can grab and rearrange.
⠿
⚙
The active block. Bold purple accent border, card elevates slightly. Clear visual feedback — you know exactly where you are.

    print("world")

B — Defined & Tactile

Blocks always have a faint card background. You always see boundaries. Hover lifts the card with a shadow. Active gets a bold accent. Feels like dragging tangible cards. More structured, less minimal.
